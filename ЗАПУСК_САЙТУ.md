# 🌸 КВІТКОВИЙ МАГАЗИН - ІНСТРУКЦІЯ ЗАПУСКУ

## 🚀 ШВИДКИЙ ЗАПУСК (РЕКОМЕНДОВАНО)

### ⚡ Подвійний клік на файл:
```
START_SERVER.cmd
```

**✅ Це найпростіший спосіб запуску!**

### 🔧 Альтернативні способи:

```cmd
python manage.py runserver
```

**АБО через PowerShell:**

```powershell
.\START_SERVER.cmd
```
